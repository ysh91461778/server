[프로젝트] 수학 학원 관리/학습 웹앱

[핵심 구조]
- Backend: Flask (Python 3.10+), JSON 파일 스토리지 (students, videos, progress, logs, tests, materials, etc.)
- Frontend: static/js (admin / student / tests), 다크 모드 지원
- 주요 데이터 규칙:
  • weekend-slots: 토/일 slot=1~3 또는 배열 [1,2]
  • today_order, extra-attend, weekend-slots: sid는 문자열 키
  • 자유의 몸: archived=false 학생 전체 (날짜 무관)
  • 완료 표시: logs[date][sid].done === true
  • 테스트 이름 매칭: 공백 제거+lowercase 정규화 (단원평가 + FINAL + HELL)

[주요 페이지/JS]
- /admin : today.js (오늘표), calendar.js (캘린더), clinicModal.js (클리닉), exportLogs.js (TXT 내보내기)
- /students : all.html + all.js (전체 학생)
- /student/<sid> : student.js (학생별 페이지)
- /tests : tests.html + tests.js (테스트 카테고리/시험/문항수 관리 + 통계)
- hoverTooltip.js : Hover시 학생 진도 + 단원평가/FINAL/HELL 상태 표시

[최근 델타 반영]
- FINAL 3회 → 문항수 24
- HoverTooltip: 파이널/헬 응시 여부도 표시, 색상 반영
- exportLogs: skip(건너뜀) 제외 후 정렬 출력
- tests 관리 페이지: 카테고리/시험 추가·삭제, 문항수 수정, 통계 표시
- 자료 업로드: 다중 파일 선택 가능
- 오늘표: 저장 후 자동 재정렬 / 결석 복구 기능 추가
- 자유의 몸: archived=false 학생 전체 출력
