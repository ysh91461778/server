테스트 관리 / 통계

커리큘럼별로 테스트를 등록하고 정답(회차 1~3)을 세팅한 뒤, 학생이 입력한 답안을 채점하여 통계를 봅니다.

테스트 목록

종류/커리 선택 → 목록에서 시험 선택 → 시험명/문항수/정답(3줄) 수정 후 저장
종류
커리
선택된 종류/커리에 속한 시험 목록
레벨/난이도 시험을 분리하려면 이름을 다르게(예: L3/L4/L5/L6) 해두면 통계가 섞이지 않습니다.
정답도 문항 수만큼 맞춰 저장됩니다.
· 정답은 1 3 4 2 / 1,3,4,2 / 1342 전부 가능
· 3줄이면 각각 1회차/2회차/3회차로 저장됩니다.
· 1줄만 입력하면 1~3회차 모두 동일 정답으로 저장됩니다.
